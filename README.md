# wy-image-viewer
仿微信`previewImage`，用来展示一个或者一组图片
## 特点：

1. 支持单张图片展示
2. 支持多张图片展示
3. 支持对展示的图片放大缩小
4. 支持双击图片放大缩小
5. 支持上下拖动某图片关闭Viewer
6. 支持对放大的图片进行拖动

## 安装方式
`npm install image-shower@latest`

## 使用方式
具体请求查看`demo/src`中的示例
